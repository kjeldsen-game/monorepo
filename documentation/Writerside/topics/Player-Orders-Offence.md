# Player Orders - Offence

<primary-label ref="wip"/>

## Shoot
- **Effect:** Standard shot on goal using either **SC** (Shooting) or the **Aerial equivalent**

## Dribble
- **Effect:** Attempts to dribble past the next defender
    - No penalty or bonus applied

## Pass
- **Effect:** Attempts to pass to another attacker in the attacking zone

## Shoot Straight
- **Effect:** A direct shot with **no control duel**
    - Shot quality is calculated as:
      ```
      (SC + BC) / 2
      ```  
      where **SC** is Shooting ability and **BC** is Ball Control
